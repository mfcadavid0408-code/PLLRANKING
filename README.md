Este proyecto nació en **CodePen** como mi primera práctica de HTML y CSS.  
Decidí trasladarlo a GitHub para compartirlo, mejorarlo y publicarlo con **GitHub Pages**.  


Es una página web inspirada en la serie *Pretty Little Liars*, donde presento un ranking de los personajes que han sido parte del misterio de “A”.  
El diseño combina colores intensos y una estructura sencilla para practicar conceptos básicos de desarrollo web.  

- HTML5
- CSS3
- Bootstrap (para la barra de navegación y estilos responsivos)
